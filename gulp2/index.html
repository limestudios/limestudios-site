> About Lime Studios

Lime Studios is an independent game development and design studio created back in 2009 by Liam Craver. Lime Studios' current game development project is called Project Arrhythmia, a rhythmic dodge-em up.

Project Arrhythmia will be coming to the WiiU, PC and Mac.

> Liam Craver [Owner]

My name is Liam Craver; I am a developer and a designer from New Orleans, Louisiana. My passion is technology; I'm always eager to learn about the latest gadgets or the new features that are in the latest patches of my favorite software.

I am an alumnus of New Orleans Center for the Creative Arts and Benjamin Franklin Highschool; currently I am pursuing a computer science degree at the University of Louisiana at Lafayette.

I work freelance when I can during the school year, but especially during the summer; if you want, check out my projects page for examples of my work. I'm open to working on projects of all sorts!

Most of the time, I am a pretty ordinary person. I say things on Twitter and Tumblr. My code ends up on GitHub most of the time.

The purpose of this website is to serve as an online repository for all my independent work and work done under Lime Studios. The code style guide document, the processes used to build it, and the source can be found on GitHub.

If you have any questions or feedback, you may want to email me at liam@limestudios.net —which you should definitely do since I love to talk.
